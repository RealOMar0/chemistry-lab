# Chemistry Lab - تجارب الكيمياء التحليلية

موقع تفاعلي لتعليم تجارب الكشف عن الأنيونات والكاتيونات في الكيمياء التحليلية.

## المميزات

- 🟢 **الكشف عن الأنيونات (الشقوق الحامضية)** - 7 أنيونات مختلفة
- 🔵 **الكشف عن الكاتيونات (المجموعة 1-3)** - 6 كاتيونات
- 🟣 **الكشف عن الكاتيونات (المجموعة 4-6)** - 6 كاتيونات
- ❓ **ملح مجهول المجموعة** - تحديد الملح من خلال التجارب

## الكواشف المتاحة

### الأنيونات
- dilute HCl
- H₂SO₄ مركز
- Potassium chromate
- FeCl₃
- MgSO₄
- BaCl₂
- خلات الرصاص

### الكاتيونات 1-3
- dilute HCl
- NH₄Cl + NH₄OH (in excess)
- H₂S Soln
- Potassium chromate
- NH₄OH
- NaOH
- dilute acetic acid + H₂S

### الكاتيونات 4-6
- NH₄Cl + NH₄OH (in excess) + H₂S
- NaOH
- Potassium chromate
- NH₄Cl + NH₄OH (in excess) + (NH₄)₂CO₃
- Sodium Cobalti nitrite

## الاستخدام

1. اختر القسم المطلوب من القائمة الرئيسية
2. اختر ملح معين أو اختر عشوائياً
3. جرب الكواشف المختلفة لاكتشاف الملح
4. استخدم التلميحات إذا احتجت للمساعدة
5. خمّن الملح أو اكتشفه من خلال التفاعلات
